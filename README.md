
# Stationary Shop – PHP Website

A simple stationary shop website built with PHP and MySQL.

Features
- Home page with product listing
- Product categories
- Shopping cart
- Admin panel to manage products
- Contact page
- Fully responsive design

Technologies
- PHP (Core PHP)
- MySQL
- HTML5, CSS3
- Bootstrap 5
- JavaScript
- 
Note
This project uses MySQL database – make sure to import the .sql file.
